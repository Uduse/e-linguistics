The ELTK (e-Linguistics Toolkit) is a suite of open source
Python modules supporting research and development in processing 
linguistics data especially data that a field linguist might collect
and annotate and especially in the context of the Semantic Web. 

Documentation: 
    Documentation can be found at http://purl.org/linguistics/eltk
